TMT Maldives
===================
Take me to Maldives Pvt. Ltd more commonly referred as ''tmtMaldives'' is a local travel operator based in Maldives.
    
#### Install Gulp:
    $ npm install --save-dev gulp
        
#### Install Server-side Dependencies
    $ npm install

#### Install Bower
    $ npm install -g bower
    
#### Run Gulp Tasks
    $ gulp
    $ gulp watch
